## Architecture

This is an **Astro 5** static site for Reina Multiservicios, a Spanish industrial cleaning company. The site is in Spanish and targets `https://reinamultiservicios.es`.

### Key Technologies
- **Astro v5** with strict TypeScript
- **Tailwind CSS v4** + **DaisyUI v5** (themes: `bumblebee` / `dracula`)
- **@astrojs/mdx** for service content
- **@astrojs/sitemap** (excludes `/legal/*` and `/gracias`)
- **astro-swiper** for hero slider, **astro-lightgallery** + **photoswipe** for image galleries

### Content Collections
The only collection is `services` (`src/content/services/*.mdx`). Each service file defines its own page data via frontmatter (title, h1, subtitle, headerImage, features, gallery, faqs, parallaxImage, slogan). Dynamic routes are generated by `src/pages/servicios/[slug].astro`.

### Data Layer
Static company/site data lives in `src/data/`:
- `company.ts` — business contact info (phone, email, WhatsApp, address, CIF)
- `siteMap.ts` — navigation structure with TypeScript interfaces
- `homeFaqs.ts` — FAQ entries for the homepage

### Routing
```
/                   → pages/index.astro
/servicios/         → pages/servicios/index.astro
/servicios/[slug]/  → pages/servicios/[slug].astro (from content collection)
/contacto/          → pages/contacto.astro
/sobre-nosotros/    → pages/sobre-nosotros.astro
/gracias/           → pages/gracias.astro (post-contact redirect, excluded from sitemap)
/legal/*            → pages/legal/ (excluded from sitemap)
```

### Styles
Global CSS is in `src/styles/global.css` using Tailwind v4 syntax (`@import "tailwindcss"`). Custom animations (`fadeUp`, `kenburns`) and reusable classes (`.main-header`, `.main-container`) are defined there. Images use a `data-fade` attribute for fade-in effects.

### Icons
SVG icons are imported directly as Astro components from `src/icons/`.
